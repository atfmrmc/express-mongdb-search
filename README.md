# express-mongdb-search

Run containers command:

```
docker-compose up -d --watch
```

Environment variables:

```
MONGODB_ROOT_USER=admin
MONGODB_ROOT_PASSWORD=adminpassword
MONGODB_USER=user
MONGODB_PASSWORD=userpassword
MONGODB_HOST=mongodb
MONGODB_PORT=27017
MONGODB_DB=bibliotheque
PORT=3000
JWT_SECRET=secret
JWT_LIFETIME=30d
```

_(Default values for development environment)_
